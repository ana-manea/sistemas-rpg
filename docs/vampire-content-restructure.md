# Reorganização do conteúdo de Vampiro

As páginas HTML enviadas foram incorporadas como páginas normais do sistema `vampire`, sem o conceito de "páginas recuperadas".

## Páginas com conteúdo importado

- Homepage
- Conceitos
- Sociedade dos Membros
- Camarilla
- Anarquistas
- Sabá
- Clãs
- Regras
- Criação de Personagens
- Disciplinas
- Vampiros
- Sistemas Avançados
- Cidades
- Ferramentas
- Camadas de Conhecimento

## Estrutura

Cada página agora possui uma pasta própria em `frontend/src/systems/vampire/pages/`.
O conteúdo principal fica em `frontend/src/systems/vampire/content/articles.js`, usado pelos componentes de página.

Páginas planejadas sem conteúdo foram criadas como páginas de construção, prontas para receber conteúdo futuramente.
